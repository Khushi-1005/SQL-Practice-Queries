# SQL Practice Queries

## Project Overview

This repository contains SQL practice queries demonstrating fundamental SQL concepts using PostgreSQL.

## Topics Covered

* SELECT Statements
* WHERE Clause
* ORDER BY
* DISTINCT
* Aggregate Functions:
  COUNT(), SUM(), AVG(), MAX(), MIN()
* GROUP BY
* HAVING Clause
* LIKE Operator
* BETWEEN Operator

## Database

PostgreSQL

## Skills Demonstrated

* Data Retrieval
* Data Filtering
* Data Aggregation
* Grouping and Reporting
* SQL Query Writing
